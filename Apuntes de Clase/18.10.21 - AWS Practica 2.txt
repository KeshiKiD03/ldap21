18.10.21 - AWS Practica 2
-------------------------

- Iniciar Docker desde inicio para siempre

1. Crear una VM Debian AWS e instalar los paquetes por defecto.
	Incluir el cliente ldap
	
Fer el desplegament d’un container LDAP detach en una VM de AWS EC2

Part 1

    Crear una VM debian a aws i instal·lar-hi  els paquets per defecte més usuals. Incloure el client de ldap.

    Instal3lar-hi també docker.

    Desplegar el servidor ldap

    Desplegar el servidor phpldapadmin

    Crear un security group anomenat ldap-phpldapadmin que permeti el tràfic als ports 22, 80 i 389.

    Verificar que teniu accés des del host de l’aula al servei ldap i al servei phpldapadmin.


Part 2

    Configurar la màquina aws per tal de que el servei docker estigui engegat automàticament en arrancar.

    Configurar els dos serveis (ldap i phpldapadmin) perquè el docker engegi els dos containers auomaticament en iniciar-se.

    Generar una imatge base AMI de la imatge que heu fabricat.

    Generar una nova VM basada en la vostra AMI
